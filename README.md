 Hotel Management System (Python)

 Project Overview

This is a Python-based Hotel Management System designed to manage hotel operations such as room booking, checkout, billing, and customer search. The system uses JSON for persistent data storage and follows a modular programming approach.



Features

* View available and occupied rooms
* Book rooms with customer details
* Checkout system with bill calculation
* Search customer by name
* Persistent data storage using JSON
* Real-time room status updates


Technologies Used

* Python
* JSON (for data storage)
* File Handling
* Modular Programming


Project Structure

hotel_management_system/
│
├── main.py          # Main control file (menu system)
├── rooms.py         # Room data handling
├── booking.py       # Booking logic
├── billing.py       # Checkout & billing
├── search.py        # Customer search feature
├── rooms.json       # Data storage
└── README.md        # Project documentation
```


⚙️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/hotel-management-system-python.git
```

2. Navigate to the folder:

```
cd hotel-management-system-python
```

3. Run the program:

```
python3 main.py
```
